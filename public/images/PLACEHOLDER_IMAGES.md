## Image Specifications

- **Format**: JPG for photos
- **Optimization**: Compress images to < 500KB each for performance
- **Quality**: JPEG compression at ~80%
- **Aspect Ratios**: 
  - Hero: 16:9
  - Villas: 3:2
  - Experience: 4:3
- **Alt Text**: All images will have descriptive alt text for accessibility

## Placeholder Notes

Currently, these are placeholders. Replace with actual images when available.
If images are missing, the site will gracefully fallback to a placeholder state.

## Image Optimization Tips

1. Use tools like TinyPNG or ImageOptim before adding images
2. For Photoshop: Export > Save for Web (Legacy) at 80% quality
3. For Mac: Use Squoosh.app or ImageOptim
4. For Windows: Use TinyPNG desktop app or FileOptimizer
5. Aim for all images combined to be under 5MB total
